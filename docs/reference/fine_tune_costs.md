# Internal function to fine-tune cost parameters

Internal function to fine-tune cost parameters

## Usage

``` r
fine_tune_costs(log_cost_mult, target, fauna, fleets, years = 25, tune_type)
```

## Arguments

- log_cost_mult:

  log cost per unit effort multiplier

- target:

  the target to be achieved

- fauna:

  a list of fauna

- fleets:

  a list of fleets

- years:

  the number of years to run the optimization

- tune_type:

  what is being tuned for

## Value

fine-tuned costs
