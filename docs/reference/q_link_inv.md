# Inverse link function mapping (0, 1) back to positive reals

Applies the transformation `raw_q = q / (1 - q)`, the inverse of
[`q_link`](https://danovando.github.io/marlin/reference/q_link.md).

## Usage

``` r
q_link_inv(q)
```

## Arguments

- q:

  Numeric in (0, 1).

## Value

Positive numeric.
